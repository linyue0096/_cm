# 夏農-哈特利定理 (Shannon–Hartley Theorem)
* 定義:
它量化了在一個受高斯白噪聲 (AWGN) 干擾的連續類比信道中，理論上每秒最多能傳輸多少位元（Bits）。
* 數學公式:
$$C = B \log_2 \left( 1 + \frac{S}{N} \right)$$
* $C$：信道容量 (Capacity)，單位 bits/s。
* $B$：頻寬 (Bandwidth)，單位 Hz。
* $S/N$：訊號與噪聲的功率比 (SNR)，注意這是線性比值，不是 dB。
數學推導邏輯
公式是由兩個物理限制乘積而成的：
$$\text{容量 } C = (\text{每秒能送的樣本數}) \times (\text{每個樣本能攜帶的位元數})$$
1. 每秒樣本數 (與 B 有關)根據奈奎斯特採樣定理 (Nyquist Rate)，如果一個信號的頻寬是 $B$，我們每秒最多可以取樣 $2B$ 次而不遺失資訊。
$$\text{Samples per second} = 2B$$
2. 每個樣本的資訊量(與 S/N 有關)一個樣本能攜帶多少資訊，取決於我們能分辨出多少個不同的電壓準位 (Levels)。
* 信號+噪聲的總振幅：正比於 $\sqrt{S+N}$。
* 噪聲的干擾半徑：正比於 $\sqrt{N}$（這是分辨率的極限，小於這個範圍會被噪聲混淆）。
- 可分辨的狀態總數($M$)為：
$$M = \frac{\text{總振幅}}{\text{噪聲振幅}} = \frac{\sqrt{S+N}}{\sqrt{N}} = \sqrt{1 + \frac{S}{N}}$$
- 將狀態數轉換成位元數
($\log_2 M$)：$$\text{Bits per sample} = \log_2 \left( \sqrt{1 + \frac{S}{N}} \right) = \frac{1}{2} \log_2 \left( 1 + \frac{S}{N} \right)$$
- 結論將兩者相乘：
$$C = (2B) \times \left[ \frac{1}{2} \log_2 \left( 1 + \frac{S}{N} \right) \right] = B \log_2 \left( 1 + \frac{S}{N} \right)$$

# 夏農信道編碼定理
* 定義:
一個存在性定理。它定義了「通信的極限」。它證明了只要傳輸速率低於容量 $C$，我們就一定能找到某種編碼方法，讓錯誤率降到任意低。
* 數學描述 (機率極限):
對於一個離散無記憶信道 (DMC)，信道容量定義為互信息 (Mutual Information) 的最大值
$$C = \max_{p(x)} I(X; Y)$$
1. 定理表示對於任意傳輸速率 $R$ 和區塊長度 $n$：正向定理 (Achievability)：若 $R < C$，則當 $n \to \infty$ 時，存在一種編碼使得錯誤機率 $P_e \to 0$。
2. 逆定理 (Converse)：若 $R > C$，則當 $n \to \infty$ 時，錯誤機率 $P_e \to 1$ (必然出錯)。
* 數學直覺 (球體堆積 Sphere Packing)我們可以將這個定理用高維幾何的[球體堆積]概念來理解：
1. 高維空間：想像一個 $n$ 維的空間，代表所有可能的接收信號。
2. 碼字 (Codeword)：我們選定的標準信號是這個空間中的一個「點」。
3. 噪聲球 (Noise Sphere)：因為有噪聲，發送一個點後，接收到的信號會落在該點周圍的一個「球體」內。
